### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code recommended)
- Live Server extension (for local development)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Dilawarkhandeveloper/Color-Palette-Generator.git
   cd Color-Palette-Generator
   ```

2. **Open the project:**
   - Open the project folder in VS Code or your preferred editor

3. **Launch Live Server:**
   - Make sure you have the Live Server extension installed
   - Right-click `index.html` and select "Open with Live Server"
   - Or simply open `index.html` in your browser

### Alternative Setup

You can also run this project by simply opening the `index.html` file directly in your browser - no server required!

## âš™ï¸ How to Use

1. **Generate Palettes:**
   - Click the "Generate Palette" button to create new colors
   - Five random colors will appear instantly

2. **Lock Colors:**
   - Click the lock icon on any color to preserve it
   - Locked colors won't change when generating new palettes

3. **Copy Hex Codes:**
   - Click any color box to copy its hex code to clipboard
   - Visual feedback confirms successful copying

4. **Save Favorites:**
   - Click "Save Palette" to add current colors to your favorites
   - Saved palettes appear in the list below

5. **Load Saved Palettes:**
   - Click on any saved palette to load it instantly
   - Delete unwanted palettes with the trash icon
